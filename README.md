# WIP


### Setup on windows
- [install python 3.11](https://www.python.org/downloads/windows/)
- [install git](https://gitforwindows.org/)


### Download code
```shell
$ git clone https://github.com/esemi/backtester.git
$ cd backtester
```


### Run backtesting tool
```bash
python -m app.backtester
```
